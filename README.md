# OpenFOAM_GUI
A graphical user interface for OpenFOAM

Features:
1. Run terminal commands with GUI
2. Syntax highlighting
3. Auto indent

System requirements:
1. Python 3.x
2. PyQt5

Running:
1. Install python3.6
sudo apt-get install python3.6

2. Install PyQt5
python3 -m pip install PyQt5

3. Run application
python3 OpenFOAM_IDE.py
